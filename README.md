# Superstore Sales Dashboard

An interactive Excel dashboard built using the Global Superstore dataset to analyze sales performance, profitability, customer segments, regional performance, and product categories.

This project demonstrates how raw business data can be transformed into an executive-level dashboard using Microsoft Excel, enabling quick decision-making through dynamic visualizations and KPI tracking.

---

## Project Objective

The objective of this project is to provide business stakeholders with an interactive dashboard that allows them to monitor key performance indicators (KPIs), identify sales trends, compare regional performance, and evaluate product profitability.

---

## Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Orders
- Profit Margin
- Average Order Value (AOV)

## Business Insights

- Technology generated the highest sales and profit.
- Consumer customers contributed the largest share of total sales.
- Standard Class was the most frequently used shipping mode.
- Regional sales performance varies significantly across states.
- Monthly sales trends help identify seasonal business patterns.
